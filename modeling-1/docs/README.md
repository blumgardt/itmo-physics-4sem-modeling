# Визуализация магнитного поля цилиндрического соленоида

## Описание работы

В данной лабораторной работе реализована программа для вычисления и визуализации магнитного поля цилиндрического соленоида с заданными параметрами. По условию требовалось, используя принцип суперпозиции магнитных полей, вычислить распределение магнитной индукции `B(x, y)` в плоскости, в которой лежит ось соленоида. Также параметры модели должны задаваться пользователем.

В программе задаются основные характеристики соленоида:

- `D` - диаметр соленоида,
- `L` - длина соленоида,
- `N` - число витков,
- `I` - сила тока.

На основе этих параметров строится визуализация распределения магнитного поля в выбранной плоскости.

---

## Физическая основа работы

С физической точки зрения соленоид представляет собой катушку из большого числа витков, по которым течёт электрический ток. Каждый виток создаёт собственное магнитное поле. Тогда полное магнитное поле соленоида можно найти как сумму магнитных полей, создаваемых всеми витками по отдельности.

Это и есть принцип суперпозиции:

`B = B1 + B2 + B3 + ... + BN`

где:

- `B1, B2, ..., BN` - поля отдельных витков,
- `N` - число витков.

Таким образом, соленоид в модели рассматривается как совокупность большого числа круговых токовых контуров, расположенных вдоль общей оси.

---

## Теоретические сведения

Основой вычисления магнитного поля является закон Био–Савара. Он описывает магнитное поле, создаваемое элементом проводника с током:

`dB = (μ0 / 4π) · (I · dl × r) / r^3`

где:

- `μ0` - магнитная постоянная,
- `I` - сила тока,
- `dl` - элемент проводника,
- `r` - радиус-вектор от элемента тока до точки наблюдения.

В программе используется значение магнитной постоянной:

- `mu0` - магнитная постоянная: `μ0 = 4π · 10^-7`.

Для одного кругового витка поле в произвольной точке выражается через эллиптические интегралы. Поэтому в программе используются математические функции для их численного вычисления. Это позволяет получить более точную модель поля, чем при использовании только приближённых школьных формул.

---

## Модель соленоида

В данной работе соленоид моделируется как набор из `N` витков, равномерно распределённых вдоль длины `L`.

Основные параметры модели:

- `D` - определяет диаметр соленоида,
- `L` - определяет его длину,
- `N` - задаёт количество витков,
- `I` - задаёт ток в витках.

Радиус соленоида определяется как:

`R = D / 2`

Чем больше число витков и чем больше ток, тем сильнее магнитное поле внутри соленоида. При увеличении длины соленоида при том же числе витков плотность витков уменьшается, поэтому поле ослабляется.

---

## Что именно показывает программа

Программа строит картину магнитного поля в плоскости, проходящей через ось соленоида. На экране отображаются:

- распределение модуля магнитной индукции,
- силовые линии магнитного поля,
- геометрические границы соленоида,
- значения поля в центре соленоида и теоретического поля для длинного соленоида.

Для этого в программе используются следующие группы параметров:

### Параметры экрана и области отображения

- `screenWidth` - ширина окна,
- `screenHeight` - высота окна,
- `scale` - масштаб изображения,
- `extZ`, `extY` - размеры области визуализации по координатам.

### Параметры расчётной сетки

- `gridNZ` - число точек сетки по оси соленоида,
- `gridNY` - число точек сетки по поперечной координате.

Чем больше размер сетки, тем точнее распределение поля, но тем больше вычислительная нагрузка.

### Параметры результата

- `bCenter` - поле в центре соленоида,
- `bTheory` - теоретическое значение поля для длинного соленоида,
- `heatmapMax` - максимальное значение поля для шкалы цветовой карты,
- `heatmapAuto` - автоматический или фиксированный режим масштабирования цветовой карты.

---

## Теоретическое значение поля

В программе дополнительно выводится теоретическая оценка магнитного поля для длинного соленоида:

`B = μ0 n I = μ0 (N / L) I`

где:

- `n = N / L` - число витков на единицу длины.

Эта формула хорошо работает для достаточно длинного соленоида, когда краевыми эффектами можно пренебречь. В реальной модели соленоид имеет конечную длину, поэтому поле в центре, вычисленное численно, может отличаться от идеального теоретического значения.

## Визуализация результатов

Для наглядности программа использует два способа отображения магнитного поля.

### 1. Цветовая карта

Модуль магнитного поля отображается в виде цветовой карты. Цвет зависит от величины поля:

- тёмные цвета соответствуют более слабому полю,
- яркие цвета соответствуют более сильному полю.

Это позволяет визуально определить, где поле максимально и как оно меняется в пространстве.

### 2. Силовые линии

Также на экран выводятся силовые линии магнитного поля. Они показывают направление вектора магнитной индукции в разных точках.

С физической точки зрения силовые линии позволяют увидеть общую структуру поля:

- внутри соленоида линии в основном направлены вдоль оси,
- снаружи они замыкаются вокруг катушки.

Такая картина соответствует классическим представлениям о магнитном поле соленоида.

## Управление программой

Во время работы приложения можно изменять параметры модели:

- `D / Shift + D` - увеличить или уменьшить диаметр,
- `L / Shift + L` - увеличить или уменьшить длину,
- `N / Shift + N` - увеличить или уменьшить число витков,
- `I / Shift + I` - увеличить или уменьшить силу тока,
- `+ / -` - изменить масштаб изображения,
- `R` - вернуть исходные параметры,
- `H` - изменить режим шкалы цветовой карты.

## Физический смысл получаемой картины

По результатам визуализации можно сделать несколько выводов.

### Поле внутри соленоида

Внутри соленоида магнитное поле оказывается наибольшим и достаточно однородным, особенно если длина соленоида больше его диаметра. Это согласуется с теорией, согласно которой внутри длинного соленоида поле почти постоянно.

### Поле у краёв

У торцов соленоида поле становится менее однородным, а силовые линии начинают искривляться. Это связано с краевыми эффектами, которые всегда возникают у соленоида конечной длины.

### Поле снаружи

Снаружи соленоида магнитное поле заметно слабее, чем внутри. Силовые линии выходят из одной области, огибают соленоид и замыкаются снаружи, что соответствует общей картине магнитного поля токовой катушки.

### Влияние параметров

Изменение параметров модели приводит к ожидаемым изменениям:

- увеличение `N` усиливает магнитное поле,
- увеличение `I` также усиливает поле,
- увеличение `L` при фиксированном `N` уменьшает плотность витков и ослабляет поле,
- изменение `D` влияет на геометрию поля и на форму силовых линий.
