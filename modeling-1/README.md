# Моделирование 1 — Магнитное поле цилиндрического соленоида

## Задание

Используя принцип суперпозиции магнитных полей, вычислить и визуализировать магнитное поле цилиндрического соленоида диаметром **D** и длиной **L** с числом витков **N**. Построить распределение **B(x, y)** в плоскости, содержащей ось соленоида.

## Физическая модель

Соленоид моделируется как **N** круговых витков с током **I**, равномерно распределённых вдоль оси от −L/2 до +L/2.

Поле каждого витка вычисляется аналитически через **закон Био-Савара**, сведённый к полным эллиптическим интегралам первого и второго рода K(k) и E(k):

$$
B_z(r, z) = \frac{\mu_0 I}{2\pi} \cdot \frac{1}{\sqrt{(R+r)^2 + z^2}} \left[ K(k^2) + \frac{R^2 - r^2 - z^2}{(R-r)^2 + z^2} \cdot E(k^2) \right]
$$

$$
B_r(r, z) = \frac{\mu_0 I}{2\pi} \cdot \frac{z}{r\sqrt{(R+r)^2 + z^2}} \left[ -K(k^2) + \frac{R^2 + r^2 + z^2}{(R-r)^2 + z^2} \cdot E(k^2) \right]
$$

где $k^2 = \frac{4Rr}{(R+r)^2 + z^2}$, а на оси ($r = 0$) используется формула:

$$
B_z = \frac{\mu_0 I R^2}{2(R^2 + z^2)^{3/2}}, \quad B_r = 0
$$

Полное поле соленоида — **суперпозиция** (сумма) полей всех N витков.

## Запуск

```bash
go run .
```

Затем открыть http://localhost:8080 в браузере.

**Зависимости:** Go 1.21+ (внешних Go-зависимостей нет, Plotly.js загружается с CDN).

go run . -port 3000

## Интерфейс

Интерактивная визуализация в браузере с:

- **Цветовой картой** |B(z, y)| — распределение модуля магнитной индукции в плоскости оси соленоида (colormap Inferno)
- **Линиями поля** — визуализация направления вектора B
- **Контуром соленоида** (голубые линии)
- **4 слайдерами** для изменения параметров в реальном времени:
  - **D** — диаметр (2–30 см)
  - **L** — длина (5–100 см)
  - **N** — число витков (5–500)
  - **I** — ток (0.1–10 А)
- **Кнопкой «Сброс»** — возврат к значениям по умолчанию

При перемещении любого слайдера поле пересчитывается и график обновляется.

## Параметры по умолчанию

| Параметр | Значение |
|----------|----------|
| D        | 10 см    |
| L        | 30 см    |
| N        | 100      |
| I        | 1.0 А    |

## Верификация

- Для бесконечного соленоида: $B = \mu_0 n I$, где $n = N/L$
- Поле в центре конечного соленоида немного меньше теоретического — это корректно
- Линии поля параллельны внутри и расходятся на концах — классическая картина
- `go test ./...` проверяет эллиптические интегралы против табличных значений и сходимость B к теоретическому пределу для длинного соленоида
